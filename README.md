# SMALL MESSAGE SERVER FORONTEND PART
### GO & Vue project


Setup project in localhost: 

1. Clone project
2. cd msgserv    
2. Is not necessary run  npm
3. All dependencies in  script cdn mode
4. Run project front on localhost:8000.
5. cd to /back
5. go build
6. ./back
7. Send messages from front page
